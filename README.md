# ATS Resume Builder SaaS

AI-powered resume optimization platform that improves ATS score and increases interview chances.

## Features
- AI Resume Optimization
- ATS Score Engine
- JD-based Tailoring
- Stripe Payments
- Dashboard & Analytics

## Tech Stack
- FastAPI
- React + Tailwind
- OpenAI API
- Stripe
- AWS (optional)

## Run Locally

### Backend
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload

### Frontend
cd frontend
npm install
npm start
